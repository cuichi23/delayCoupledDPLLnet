synctools module¶

General¶

The synctools module contains functions and class to compute the frequency and stability of globally synchronized states in a system of coupled PLLs. Three types of functions and classes are containd in the module.

  1. Functions containing the math with a non-class interface
synctools.get_sign_changes(x) Returns an integer array with the indices of the element in x before a sign change.
synctools.get_parametric_omega_curve(n, w, ...) Compute a parametric curve of the global synchronization frequency as a function of the delay.
synctools.get_omega_implicit(n, w, k, tau, h, m) Computes the global synchronization frequency for a given delay.
synctools.get_stability(n, w, k, h, m, tau, ...) Linear stability analysis of globally synchronized state.
  1. Classes that define a single PLL system and a single m-twist state
synctools.Triangle(freq) Periodic triangle signal vertically centered around 0
synctools.Square(freq, amp) Periodic square wave vertically centered around 0
synctools.PllSystem(n, w, k, tau, h, wc) System of coupled PLLs
synctools.TwistState(system, m, omega, l) Globally synchronized twist states
  1. Classes to perform a sweep over system parameters and to store a list of states
synctools.SweepFactory(n, w, k, tau, h, wc, m) Sweeps a system parameters of a coupled PLL system
synctools.FlatStateList([tsim]) Flat list of TwistStates

Example¶

# System parameters
n = 3
w = 2 * np.pi
k = 1.57
wc = 1 * w
m = 1
tau = np.linspace(0, 2, 250)
h = synctools.Triangle(1.0 / (2 * np.pi))
tsim = 0.1234


# Create a pll system and determine the possible m-twist states
pll = synctools.PllSystem(n, w, k, tau[0], h, wc)
s = pll.get_twist_state(m)

# Perform a delay sweep
sf = synctools.SweepFactory(n, w, k, tau, h, wc, m, tsim=tsim)
fsl = sf.sweep()

# Perform a delay sweep
sf = synctools.SweepFactory(n, w, k, tau, h, wc, m)
fsl = sf.sweep()

# Extract variables from the sweep
para_mat = fsl.get_parameter_matrix()
tau2 = fsl.get_tau()
omega2 = fsl.get_omega()
l2 = fsl.get_l()

synctools.get_stability(n, w, k, h, m, tau, omega, wc)¶

Linear stability analysis of globally synchronized state.

The computation is based on finding the roots of the two-dimensional characteristic equation. For each eigenvalue of the coupling matrix there is a different characteristic equation. Here, only the complex exponent with the largest real value of all characteristic equations is returned. Moreover, the characteristic equation associated with the eigenvector (a, a, a, ...) is excluded since it corresponds to a global phase shift of all oscillators and does not affect synchronization.
Parameters:
  • n (int) – number of oscillators
  • w (float) – intrinsic angular frequency of the oscillators
  • k (float) – coupling constant
  • tau (float) – delay
  • h (callable) – coupling function
  • m (int) – twist number
  • wc (float) – angular cut-off frequency of the low-pass filter
Returns:

lambda_nu – the complex linear stability analysis exponent with the biggest real value
Return type:

complex
